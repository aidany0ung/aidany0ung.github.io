Adding Species and Rot State

In this version, any cell that is set from alive to dead is set first to a more inert rotting state, in which it is not alive, but also cannot be updated in the same way a dead cell can be. Additionally, I add two different species. A cell of the same species counts as a living cell, but a cell of a different species does not. This two additions reduce the entropy of the system, causing a more boring equilibrium faster.
